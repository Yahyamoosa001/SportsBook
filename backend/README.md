# QuickCourt Backend

Environment variables required:

- `PORT` (default 3001)
- `MONGO_URI` (e.g., mongodb://127.0.0.1:27017/quickcourt_db)
- `JWT_SECRET`
- `CLIENT_ORIGIN` (e.g., http://localhost:8081)

Scripts:

- `npm run dev` to start with nodemon
- `npm start` to start normally

API base path: `/api`
